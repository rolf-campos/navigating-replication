# Results Map

This document lists the reproducible outputs generated by the replication notebooks.

## Notebook 1: `1_steady_states.ipynb`
### Outputs
#### Steady-State Data Files (`ss/`)
- `ss_hank_ini.json` – Baseline steady state
- `ss_hank_end.json` – Steady state after increasing government debt
- `ss_hank_end_no_reaction.json` – Steady state with no policy reaction

#### Table
- `results/table_4.tex` — Table 4 in the paper

Computes baseline and alternative steady states and generates the comparison table used in the main text.
